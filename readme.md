#Wireless-Ethernet
Instructions:
Run ethernet_gui.py, the rest are supporting files not meant to run individually.
Notes:
To have the code work you need to have an ADALM-PLUTO SDR plugged in to your computer, default settings.
This code has only successfully been tested on Windows 11.
The radios require about 5 minutes to warm up - do not expect messages, especially ethernet, to go through until the radios have had time to warm up.
